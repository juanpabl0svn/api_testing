Order Algorithms


Welcome to order algorithms, here you will be able to order a public database, which you can see in the path next path Database, you will also be able to order these registers using this algoritms and giving as a parameter an atribute of the database, here are some examples Bubble sort, Quick sort, Merge sort,Counting sort, Radix sort,you can use whichever you want writting in the current path "/ORDER_ALGORITHM/ATTRIBUTE", if the attribure is NaN or it contains constraits in the algoritm (decimals or negative number) it will throw an error, thanks for reading this and feel free to use this API
